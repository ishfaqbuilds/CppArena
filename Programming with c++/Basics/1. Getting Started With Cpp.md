
#### 1. What is C++?

**History:** C++ was developed by Bjarne Stroustrup at Bell Labs starting in 1979 as an extension of the C programming language. Originally called "C with Classes," it was renamed C++ in 1983. The "++" operator in C means "increment," symbolizing an enhanced version of C.

**Key Features:**

- **Object-Oriented Programming (OOP)**: Supports classes, objects, inheritance, polymorphism, and encapsulation
- **Low-level control**: Direct memory management and hardware access
- **High performance**: Compiled language with minimal runtime overhead
- **Multi-paradigm**: Supports procedural, object-oriented, and generic programming
- **Rich standard library**: Extensive collection of pre-written functions and classes
- **Platform independence**: Code can run on multiple operating systems
- **Strong type system**: Helps catch errors at compile time

#### 3. Your First Program (Hello World)

```cpp
#include <iostream> 
using namespace std;
int main() 
{ 
	cout << "Hello, World!" << endl; 
	return 0; 
}
```

#### 4. Compiling and Running

##### Using Command Line (g++):

```cpp
# Compile
g++ hello.cpp -o hello

# Run (Windows)
hello.exe

# Run (Linux/macOS)
./hello
```

#### Using IDE:

Most IDEs have a "Build" or "Run" button that compiles and executes your program automatically.

#### 5. Basic Program Structure

Let's break down the Hello World program:

```cpp
#include <iostream>    // Preprocessor directive
                       // Includes input/output stream library

int main() // Main function - program entry point
{          // 'int' means it returns an integer
    
    std::cout << "Hello, World!" << std::endl;
    // std::cout - standard character output stream
    // << - insertion operator
    // std::endl - end line and flush buffer
    
    return 0;         // Return 0 to indicate successful execution
}                     // End of main function
```

#### Key Components:

##### # include directives:

- Tell preprocessor to include header files
- `<iostream>` provides input/output functionality
- Must appear at the top of the file

##### main( ) function:

- Every C++ program must have exactly one main() function
- Program execution starts here
- `int` return type indicates success (0) or failure (non-zero)

##### Statements:

- End with semicolons (;)
- Executed sequentially
- `std::cout` is used for output
- `std::cin` is used for input (when needed)

##### return statement:

- Returns a value to the operating system
- `return 0;` indicates successful program termination
