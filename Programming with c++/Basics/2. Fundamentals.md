
## 1. Variables and Constants
### Variables
Variables are containers that store data values. They must be declared before use.

Syntax:
```cpp
data_type variable_name = initial_value;

// Example:
int age = 25;
float height = 5.9;
char grade = 'A';
string name = "John";
```

**Variable Naming Rules:**

- Must start with letter or underscore 
```cpp
int age;       // OK
int _count;    // OK
int 2ndValue;  // ❌ Error: cannot start with digit
```

- Can contain letters, digits, and underscores
```cpp
int myVariable1;   // OK
int data_2025;     // OK
int score$;        // ❌ Error: $ not allowed in standard C++
```

- Cannot be C++ keywords
```cpp
int int;     // ❌ Error: 'int' is a keyword
int while;   // ❌ Error: 'while' is a keyword
int value;   // OK
```

- Case-sensitive (age ≠ Age)
```cpp
int age = 20;
int Age = 30;

cout << age; // Prints 20
cout << Age; // Prints 30 (different variable)
```

### Constants

1. Using `const` keyword (Recommended)
```cpp
const int MAX_STUDENTS = 50;
const double PI = 3.14159;
const char GRADE_A = 'A';

// const variables must be initialized when declared
// const int x;  // ERROR: const must be initialized
```

2. Using `#define` preprocessor directive (MACRO)
```cpp
#define MAX_SIZE 100
#define PI 3.14159
#define COMPANY_NAME "TechCorp"

// No semicolon needed
// Replaced by preprocessor before compilation
```

## 2. Data Types
### Primary Data Types

#### Integer Types
```cpp
int age = 25;                   // Typically 4 bytes, range: -2³¹ to 2³¹-1
short marks = 95;               // Typically 2 bytes, range: -32,768 to 32,767
long population = 1000000L;     // Typically 8 bytes, larger range
long long bigNum = 123456789LL; // At least 8 bytes
```

#### Floating Point Types
```cpp
float price = 19.99f;                    // 4 bytes, ~7 decimal digits precision
double salary = 50000.50;                // 8 bytes, ~15 decimal digits precision
long double precise = 3.141592653589793; // Extended precision
```

#### Character Type
```cpp
char letter = 'A';      // 1 byte, stores single character
char newline = '\n';    // Escape sequences
char tab = '\t';
```

#### Boolean Type
```cpp
bool isActive = true;   // 1 byte, stores true or false
bool isComplete = false;
```

#### Void Type
```cpp
void functionName() {   // No return value
    // function body
}
// Cannot declare variables of void type
// void x; // ERROR
```

### Data Type Sizes (Typical)

|Type|Size|Range|
|---|---|---|
|bool|1 byte|true/false|
|char|1 byte|-128 to 127|
|short|2 bytes|-32,768 to 32,767|
|int|4 bytes|-2,147,483,648 to 2,147,483,647|
|long|8 bytes|Very large range|
|float|4 bytes|±3.4E±38 (7 digits)|
|double|8 bytes|±1.7E±308 (15 digits)|
## 3. Type Modifiers
Type modifiers alter the meaning of base data types.
### Signed vs Unsigned
```cpp
// Signed (default) - can store positive and negative values
signed int temperature = -10;
int score = 95;  // signed by default

// Unsigned - only positive values, doubles positive range
unsigned int age = 25;
unsigned char grade = 200;  // 0 to 255 instead of -128 to 127
```

### Short and Long
```cpp
short int marks = 450;                       // Or just: short marks = 450;
long int distance = 1000000L;                // Or just: long distance = 1000000L;
long long bigNumber = 123456789123456789LL;

// Size hierarchy: short ≤ int ≤ long ≤ long long
```

### Combined Modifiers
```cpp
unsigned short age = 25;               // 0 to 65,535
unsigned long population = 5000000UL;
signed char temperature = -40;         // Explicitly signed
```

### Memory Layout Example:
```cpp
cout << "Size of int: " << sizeof(int) << " bytes" << endl;
cout << "Size of short: " << sizeof(short) << " bytes" << endl;
cout << "Size of long: " << sizeof(long) << " bytes" << endl;
cout << "Size of unsigned int: " << sizeof(unsigned int) << " bytes" << endl;
```

### 4. Input/Output (cin, cout)

Output with `cout`
```cpp
#include <iostream>
using namespace std;

int main() 
{
    int age = 25;
    string name = "Alice";
    
    cout << "Hello World!" << endl;
    cout << "Name: " << name << endl;
    cout << "Age: " << age << endl;
    
    // Chaining output
    cout << "Name: " << name << ", Age: " << age << endl;
    
    return 0;
}
```

Input with `cin`
```cpp
#include <iostream>
using namespace std;

int main() 
{
    string name;
    int age;
    float height;
    
    cout << "Enter your name: ";
    cin >> name;  // Stops at whitespace
    
    cout << "Enter your age: ";
    cin >> age;
    
    cout << "Enter your height: ";
    cin >> height;
    
    cout << "Hello " << name << "! You are " << age << " years old." << endl;
    
    return 0;
}
```

### Advanced Input/Output
```cpp
#include <iostream>
#include <string>
using namespace std;

int main() 
{
    string fullName;
    
    // Reading entire line (including spaces)
    cout << "Enter full name: ";

```

## 6. Type Conversion and Casting

- Implicit Type Conversion (Automatic)
```cpp
int main() 
{
    int x = 10;
    double y = 3.14;
    
    double result = x + y;  // int automatically converted to double
    cout << result << endl;  // Output: 13.14
    
    // Promotion hierarchy (smaller to larger):
    // char → short → int → long → float → double → long double
    
    return 0;
}
```

- Explicit Type Conversion (Manual Casting)
```cpp
int main() 
{
    double pi = 3.14159;
    int intPi = (int)pi;  // Truncates to 3
    
    int a = 7, b = 3;
    double division = (double)a / b;  // 2.33333 instead of 2
    
    cout << "Pi as int: " << intPi << endl;
    cout << "Division: " << division << endl;
    
    return 0;
}
```

#### Common Conversion Scenarios

String to Number
```cpp
#include <string>
#include <iostream>
using namespace std;

int main() 
{
    string str = "123";
    int num = stoi(str);            // String to integer
    
    string floatStr = "19.99";
    double price = stod(floatStr);  // String to double
    
    cout << "Number: " << num << endl;
    cout << "Price: " << price << endl;
    
    return 0;
}
```

Number to String
```cpp
#include <string>
#include <iostream>
using namespace std;

int main() 
{
    int age = 25;
    string ageStr = to_string(age);
    
    double price = 19.99;
    string priceStr = to_string(price);
    
    cout << "Age as string: " << ageStr << endl;
    cout << "Price as string: " << priceStr << endl;
    
    return 0;
}
```
