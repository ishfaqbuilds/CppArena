
### 1. Conditional Statements

#### Basic if-else

```cpp
if (condition) 
{
    // execute if true
} 
else if (anotherCondition) 
{
    // execute if second condition true
} 
else 
{
    // execute if all false
}
```

**Example:**

```cpp
int score = 85;
if (score >= 90) 
{
    cout << "Grade A";
} 
else if (score >= 80) 
{
    cout << "Grade B";
} 
else 
{
    cout << "Grade C or below";
}
```

### 2. Switch Statement

```cpp
switch (variable) 
{
    case value1:
        // code
        break;
    case value2:
        // code
        break;
    default:
        // default code
        break;
}
```

**Example:**

```cpp
char grade = 'B';
switch (grade) 
{
    case 'A':
        cout << "Excellent!";
        break;
    case 'B':
        cout << "Good!";
        break;
    default:
        cout << "Keep trying!";
        break;
}
```

### 3. Loops

#### for Loop (Known iterations)

```cpp
for (initialization; condition; increment) 
{
    // code to repeat
}
```

**Example:**

```cpp
for (int i = 1; i <= 5; i++) 
{
    cout << i << " ";  // Prints: 1 2 3 4 5
}
```

#### while Loop (Condition-based)

```cpp
while (condition) 
{
    // code to repeat
    // update condition variable
}
```

**Example:**

```cpp
int count = 1;
while (count <= 3) 
{
    cout << "Count: " << count << endl;
    count++;  // Don't forget to update!
}
```

#### do-while Loop (Execute at least once)

```cpp
do {
    // code to execute
} while (condition);
```

**Example:**

```cpp
int choice;
do {
    cout << "Enter choice (1-3): ";
    cin >> choice;
} while (choice < 1 || choice > 3);
```

### 4. Loop Control

#### break - Exit loop completely

```cpp
for (int i = 1; i <= 10; i++) 
{
    if (i == 5) 
    {
        break;  // Exits loop when i equals 5
    }
    cout << i << " ";  // Prints: 1 2 3 4
}
```

#### continue - Skip current iteration

```cpp
for (int i = 1; i <= 5; i++) 
{
    if (i == 3) 
    {
        continue;  // Skip when i equals 3
    }
    cout << i << " ";  // Prints: 1 2 4 5
}
```

### 5. Quick Examples

#### Number Validation

```cpp
int num;
do {
    cout << "Enter positive number: ";
    cin >> num;
} while (num <= 0);
```

#### Simple Menu

```cpp
int choice;
do {
    cout << "1. Option A\n2. Option B\n3. Exit\nChoice: ";
    cin >> choice;
    
    switch (choice) 
    {
        case 1: cout << "You chose A\n"; break;
        case 2: cout << "You chose B\n"; break;
        case 3: cout << "Goodbye!\n"; break;
        default: cout << "Invalid choice!\n";
    }
} while (choice != 3);
```

#### Pattern Printing

```cpp
// Right triangle
for (int i = 1; i <= 4; i++) 
{
    for (int j = 1; j <= i; j++) 
    {
        cout << "* ";
    }
    cout << endl;
}
// Output:
// *
// * *
// * * *
// * * * *
```
#### When to Use What?

|Structure|Use When|
|---|---|
|**if-else**|Complex conditions, ranges, boolean logic|
|**switch**|Single variable against multiple constants|
|**for**|Known number of iterations|
|**while**|Condition-based repetition|
|**do-while**|Must execute at least once|
|**break**|Exit loop early|
|**continue**|Skip current iteration|
#### Key Tips

✅ **Always initialize loop variables**  
✅ **Update condition variables in while loops**  
✅ **Use break in switch cases**  
✅ **Validate user input with loops**  
✅ **Choose the right loop type for the job**

❌ **Avoid infinite loops**  
❌ **Don't forget break in switch**  
❌ **Don't use goto (use structured alternatives)**
