## 1. Arithmetic Operators

Perform mathematical operations on numeric data.

```cpp
#include <iostream>
using namespace std;

int main() {
    int a = 15, b = 4;
    double x = 15.5, y = 4.2;
    
    // Basic arithmetic
    cout << "Addition: " << (a + b) << endl;        // 19 (Addition)
    cout << "Subtraction: " << (a - b) << endl;     // 11 (Substraction)
    cout << "Multiplication: " << (a * b) << endl;  // 60 (Multiplication)
    cout << "Division: " << (a / b) << endl;        // 3 (division) 
    cout << "Modulus: " << (a % b) << endl;         // 3 (remainder)
    
    // Float division
    cout << "Float division: "  << (x / y) << endl;  // 3.69048 
    
    return 0;
}
```

**Important Notes:**

- **Integer Division**: `15 / 4 = 3` (truncated)
- **Modulus (%)**: Only works with integers, gives remainder
- **Division by Zero**: Undefined behavior, causes runtime error

**Practical Examples:**
```cpp
// Check if number is even or odd
int number = 27;
if (number % 2 == 0) 
{
    cout << number << " is even" << endl;
} 
else 
{
    cout << number << " is odd" << endl;
}

// Calculate average
int total = 85 + 92 + 78;
int count = 3;
double average = static_cast<double>(total) / count;  // Avoid integer division
```

## 2. Assignment Operators

### Basic Assignment
```cpp
int x = 10;  // Assign 10 to x
x = 20;      // Reassign 20 to x
```

### Compound Assignment Operators
```cpp
int main() 
{
    int x = 10;
    
    x += 5;   // x = x + 5  → x becomes 15
    x -= 3;   // x = x - 3  → x becomes 12
    x *= 2;   // x = x * 2  → x becomes 24
    x /= 4;   // x = x / 4  → x becomes 6
    x %= 4;   // x = x % 4  → x becomes 2
    
    cout << "Final value of x: " << x << endl;  // 2
    
    // With other types
    string message = "Hello";
    message += " World";      // String concatenation
    cout << message << endl;  // "Hello World"
    
    return 0;
}
```

**All Assignment Operators:**

| Operator | Example   | Equivalent       |
| -------- | --------- | ---------------- |
| `=`      | `x = 5`   | Basic assignment |
| `+=`     | `x += 3`  | `x = x + 3`      |
| `-=`     | `x -= 3`  | `x = x - 3`      |
| `*=`     | `x *= 3`  | `x = x * 3`      |
| `/=`     | `x /= 3`  | `x = x / 3`      |
| `%=`     | `x %= 3`  | `x = x % 3`      |
| `&=`     | `x &= 3`  | `x = x & 3`      |
| `\|=`    | `x \|= 3` | `x = x \| 3`     |
| `^=`     | `x ^= 3`  | `x = x ^ 3`      |
| `<<=`    | `x <<= 2` | `x = x << 2`     |
| `>>=`    | `x >>= 2` | `x = x >> 2`     |

## 3. Comparison Operators

Return boolean values (true or false).

```cpp
#include <iostream>
using namespace std;

int main() {
    int a = 10, b = 20;
    
    cout << "a = " << a << ", b = " << b << endl;
    
    cout << "a == b: " << (a == b) << endl;  // false (0)
    cout << "a != b: " << (a != b) << endl;  // true (1)
    cout << "a < b: " << (a < b) << endl;    // true (1)
    cout << "a > b: " << (a > b) << endl;    // false (0)
    cout << "a <= b: " << (a <= b) << endl;  // true (1)
    cout << "a >= b: " << (a >= b) << endl;  // false (0)
    
    // String comparison
    string str1 = "Apple";
    string str2 = "Banana";
    cout << "Apple < Banana: " << (str1 < str2) << endl;  // true (lexicographic)
    
    return 0;
}
```

Common Mistakes:
```cpp
// Wrong: Assignment instead of comparison
if (x = 10)  // This assigns 10 to x, always true!
{ 
    // ...
}

// Correct: Comparison
if (x == 10) // This compares x with 10
{  
    // ...
}
```

## 4. Logical Operators

Used to combine boolean expressions.

```cpp
#include <iostream>
using namespace std;

int main() {
    bool a = true, b = false;
    int age = 25;
    double salary = 50000;
    
    // AND operator (&&) - Both conditions must be true
    cout << "true && false: " << (a && b) << endl;  // false
    cout << "true && true: " << (a && a) << endl;   // true
    
    // OR operator (||) - At least one condition must be true
    cout << "true || false: " << (a || b) << endl;  // true
    cout << "false || false: " << (b || b) << endl; // false
    
    // NOT operator (!) - Reverses the boolean value
    cout << "!true: " << (!a) << endl;   // false
    cout << "!false: " << (!b) << endl;  // true
    
    // Practical examples
    if (age >= 18 && age <= 65) {
        cout << "Eligible for employment" << endl;
    }
    
    if (salary > 40000 || age < 30) {
        cout << "Qualifies for loan" << endl;
    }
    
    if (!(age < 18)) {  // Same as age >= 18
        cout << "Adult" << endl;
    }
    
    return 0;
}
```

## 5. Increment/Decrement Operators
###### Note: Usually Helpful Incase of doing Loops
### Pre-increment and Post-increment

```cpp
#include <iostream>
using namespace std;

int main() {
    int x = 5, y = 5;
    
    // Pre-increment (++variable)
    cout << "x before: " << x << endl;           // 5
    cout << "++x returns: " << ++x << endl;      // 6 (increment first, then use)
    cout << "x after: " << x << endl;            // 6
    
    cout << endl;
    
    // Post-increment (variable++)
    cout << "y before: " << y << endl;           // 5
    cout << "y++ returns: " << y++ << endl;      // 5 (use first, then increment)
    cout << "y after: " << y << endl;            // 6
    
    return 0;
}
```

### Pre-decrement and Post-decrement

```cpp
int main() {
    int a = 10, b = 10;
    
    cout << "Pre-decrement --a: " << --a << endl;   // 9
    cout << "Post-decrement b--: " << b-- << endl;  // 10
    cout << "a is now: " << a << endl;              // 9
    cout << "b is now: " << b << endl;              // 9
    
    return 0;
}
```

## 6. Bitwise Operators

Work on individual bits of data.

```cpp
#include <iostream>
#include <bitset>
using namespace std;

int main() {
    int a = 12;  // Binary: 1100
    int b = 10;  // Binary: 1010
    
    cout << "a = " << a << " (" << bitset<4>(a) << ")" << endl;
    cout << "b = " << b << " (" << bitset<4>(b) << ")" << endl;
    
    // AND (&) - Both bits must be 1
    cout << "a & b = " << (a & b) << " (" << bitset<4>(a & b) << ")" << endl;  
    // 8 (1000)
    
    // OR (|) - At least one bit must be 1
    cout << "a | b = " << (a | b) << " (" << bitset<4>(a | b) << ")" << endl;  
    // 14 (1110)
    
    // XOR (^) - Bits must be different
    cout << "a ^ b = " << (a ^ b) << " (" << bitset<4>(a ^ b) << ")" << endl;  
    // 6 (0110)
    
    // NOT (~) - Flips all bits
    cout << "~a = " << (~a) << " (" << bitset<8>(~a) << ")" << endl;
    
    // Left shift (<<) - Multiply by 2^n
    cout << "a << 2 = " << (a << 2) << " (" << bitset<8>(a << 2) << ")" << endl;  
    // 48
    
    // Right shift (>>) - Divide by 2^n
    cout << "a >> 2 = " << (a >> 2) << " (" << bitset<8>(a >> 2) << ")" << endl;  
    // 3
    
    return 0;
}
```

## 7. Ternary Operator (?:)

Compact way to write simple if-else statements.

**Syntax:**

```
condition ? value_if_true : value_if_false
```

### Basic Usage

```cpp
#include <iostream>
using namespace std;

int main() 
{
    int age = 20;
    
    // Traditional if-else
    string status1;
    if (age >= 18) 
    {
        status1 = "Adult";
    } else 
    {
        status1 = "Minor";
    }
    
    // Ternary operator (more concise)
    string status2 = (age >= 18) ? "Adult" : "Minor";
    
    cout << "Status: " << status2 << endl;  // "Adult"
    
    // More examples
    int a = 10, b = 20;
    int max = (a > b) ? a : b;              // Find maximum
    cout << "Maximum: " << max << endl;     // 20
    
    // Nested ternary (use sparingly)
    int grade = 85;
    char letterGrade = (grade >= 90) ? 'A' : 
                      (grade >= 80) ? 'B' : 
                      (grade >= 70) ? 'C' : 'F';
    cout << "Grade: " << letterGrade << endl;  // 'B'
    
    return 0;
}
```

## 8. Operator Precedence and Associativity

**Precedence** is like the "order of operations" in math. It tells the compiler which operations to do first when you have multiple operators in one expression.
### Precedence Table (High to Low)

```cpp
// Postfix: () [] -> . ++ --
// Prefix: ++ -- + - ! ~ * & sizeof
// Multiplicative: * / %
// Additive: + -
// Shift: << >>
// Relational: < <= > >=
// Equality: == !=
// Bitwise AND: &
// Bitwise XOR: ^
// Bitwise OR: |
// Logical AND: &&
// Logical OR: ||
// Ternary: ?:
// Assignment: = += -= *= /= %= etc.
```

### Why It Matters:
```cpp
// Without understanding precedence:
int result = 5 + 3 * 2;  // You might think: (5 + 3) * 2 = 16

// With precedence knowledge:
int result = 5 + 3 * 2;  // Actually: 5 + (3 * 2) = 11
```
