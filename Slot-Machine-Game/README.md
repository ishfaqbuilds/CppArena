### Slot Machine Game (Python Terminal)

A terminal based Python slot machine game showcasing core programming concepts through interactive betting, a 3×3 grid, colored symbols, balance tracking, and win detection.

#### Features

1. Interactive 3×3 slot machine grid with randomized symbol generation
2. Colored terminal output using the colorama library for enhanced visual experience
3. Flexible betting system allowing players to bet on multiple lines with custom amounts
4. Real-time balance tracking with deposit and withdrawal mechanics
5. Automatic win detection based on matching symbols across selected lines

#### Getting Started

##### Prerequisites:

1. Python 3.7 or higher installed on your system
2. pip package manager (usually comes with Python)
3. Basic familiarity with terminal/command line operations

##### Installation:

###### 1. Clone The Repository
```
git clone https://github.com/ishfaqbuilds/Slot-Machine-Game.git
cd Slot-Machine-Game
```

###### 2. Install Dependencies
```
pip install -r requirements.txt
```

###### 3. Run The Game
```
python src/main.py
```
